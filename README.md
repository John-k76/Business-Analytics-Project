# SQL Business Analytics Project

## Project Overview
Analyzed a simulated sales database to understand revenue trends, product performance, and customer concentration using SQL and Python.

## Data
- Customers: ID, signup date, region  
- Products: ID, name, category, price  
- Orders: ID, customer, order date  
- Order Items: links products to orders, includes quantity  

## Analysis Performed
1. Monthly revenue trends  
2. Pareto analysis of customer revenue  
3. Visualizations in Python (matplotlib)  
4. Insights on revenue stability, high-value products, and top customers  

## Skills Demonstrated
- SQL (joins, aggregation, ranking, cumulative sums)  
- Python (pandas, matplotlib)  
- Analytical reasoning and business interpretation
